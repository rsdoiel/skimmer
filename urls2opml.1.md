# NAME 

urls2opml

# SYNOPSIS

urls2opml [OPTIONS] URL_FILE [URL_FILE ...]

# DESCRIPTION

urls2opml reads in one or more URL file and writes an OPML file to standard output.

# EXAMPLE

Here's of converting "myfile.urls" to "myfile.opml".


~~~shell
urls2opml myfile.urls >myfile.opml
~~~
